Class: CachingSequence

CachingSequence

new CachingSequence()

A CachingSequence is a Sequence that (probably) must fully evaluate the underlying sequence when each is called. For this reason, it provides a cache method to fully populate an array that can then be referenced internally. Frankly, I question the wisdom in this sequence type and think I will probably refactor this out in the near future. Most likely I will replace it with something like an 'IteratingSequence' which must expose a 'getIterator' and not provide get or {@link length} at all. But we'll see.
