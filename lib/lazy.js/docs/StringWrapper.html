Class: StringWrapper

StringWrapper

new StringWrapper(source)

Wraps a string exposing match and {@link split} methods that return Sequence objects instead of arrays, improving on the efficiency of JavaScript's built-in String.split and {@link String.match} methods and supporting asynchronous iteration.
