Class: AsyncSequence

AsyncSequence

new AsyncSequence(parent, interval)

An AsyncSequence iterates over its elements asynchronously when each is called.
Parameters:
Name Type Argument Description
parent Sequence A Sequence to wrap, to expose asynchronous iteration.
interval number <optional>
 How many milliseconds should elapse between each element when iterating over this sequence. If this argument is omitted, asynchronous iteration will be executed as fast as possible.
